# Changelog

## Unreleased

- Button now has type-safe text and icon-only modes. `subtle` and `destructive` remain deprecated compatibility aliases for `secondary` and `danger`.
- IconButton remains available as a deprecated compatibility wrapper; new code should use Button icon mode.
- Card link mode now exposes native anchor attributes and cannot be combined with `as`.
- ToastViewport now separately localizes visible `dismissText` and accessible `dismissLabel`.

## 0.1.0-alpha.0

Initial Nerio Core pre-release preparation.
